Ты интерпретируешь ответ пользователя в диагностическом протоколе Cognitive Identity Map v1.

Дано:
- `question_id`: идентификатор вопроса протокола.
- Ось и координата, которые уже выделены детерминированным mapper (эталон для согласованности).

Задача:
1. Кратко (2–4 предложения по-русски) отрази смысл ответа пользователя и как он соотносится с выбранной координатой.
2. В конце ответа добавь ОТДЕЛЬНОЙ строкой JSON для машинного разбора:

```json
{"axis":"goal|modality|anchor","coordinate":"<строка как у mapper>"}
```

Где `coordinate` для цели — русское название цели (Истина, Понимание, …); для модальности — буква А, Б или М; для якоря — одна кириллическая буква А–З как в вариантах вопроса.

Если пользователь уклонился от формата, всё равно предложи наиболее верную координату в JSON.
